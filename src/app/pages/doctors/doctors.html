Gestión de Equipo Médico

¿Cómo funciona el Drag and Drop?

Esta página implementa funcionalidad de arrastrar y soltar (Drag and Drop) utilizando las APIs nativas del navegador integradas con Angular. La funcionalidad permite:

  • Arrastra cualquier doctor de la lista "Doctores Disponibles" hacia la sección "Equipo Seleccionado"
  • Suelta el doctor en la zona de destino para agregarlo al equipo
  • También puedes devolver doctores del equipo seleccionado a la lista de disponibles
  • Los datos se cargan dinámicamente desde un archivo JSON usando HttpClient (equivalente a AJAX)
  • Usa los botones "×" para remover doctores individuales o "Limpiar Selección" para remover todos

Implementación técnica: Se utilizan los eventos dragstart, dragover, y drop de HTML5, manejados por métodos TypeScript del componente Angular. No se requiere jQuery ya que Angular proporciona herramientas más modernas y eficientes para este tipo de funcionalidad.
